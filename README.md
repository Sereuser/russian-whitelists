# Белый список РФ (Whitelists for Russia)

Этот репозиторий содержит актуальные списки разрешенных доменов и IP-адресов для сетевой инфраструктуры РФ, предназначенные для использования в VPN, прокси или системах фильтрации трафика. Данные собраны на основе официальных источников и предназначены для обеспечения доступа к доверенным ресурсам в условиях ограничений.

## Источники данных

- **Минцифры РФ** (mincifry.gov.ru)
- **Портал госуслуг** (gogov.ru)
- **RIPE Stat, BGPView, RADB, ipinfo** (для IP-диапазонов)
- **Последнее обновление:** 19 апреля 2026 г.

Информация собрана в поддержке Windify VPN: [https://t.me/WindifyNews](https://t.me/WindifyNews)

## Файлы в репозитории

### `domenslist.txt`
Список разрешенных FQDN (полных доменных имен) с указанием портов (обычно 443 для HTTPS, 80 для HTTP).
- Формат: `домен:порт`
- Включает поисковики (Яндекс), социальные сети (VK, OK, Mail.ru), госуслуги, банки, операторов связи, маркетплейсы и доставку.
- Примечание: Некоторые сервисы отмечены как "НЕ в официальном белом списке" (например, Sberbank, Tinkoff).

### `whitelist_ip_ranges.json`
JSON-файл с детализированной информацией по IP-диапазонам.
- Структура:
  - `meta`: Метаданные (дата генерации, общее количество префиксов и IP).
  - `services`: Массив объектов для каждого сервиса, включая:
    - `name`: Название сервиса.
    - `domain`: Основной домен.
    - `asns`: Список ASN (Autonomous System Numbers).
    - `ips_sample`: Примеры IP-адресов.
    - `prefixes`: Список CIDR-диапазонов.

### `whitelist_ip_ranges.txt`
Текстовый файл с CIDR-диапазонами, сгруппированными по сервисам.
- Формат: `CIDR-диапазон # комментарий с количеством IP`
- Включает диапазоны для Яндекса, Одноклассников, Mail.ru, Дзена и других.

## Как использовать

1. **Для VPN/прокси:** Импортируйте списки в конфигурацию для разрешения трафика только к указанным доменам/IP.
2. **Для фильтрации:** Используйте CIDR-диапазоны для настройки фаерволов или маршрутизации.
3. **Валидация:** Проверьте домен или IP на наличие в списках. Учитывайте поддомены (например, `api.example.com` валиден, если `example.com` в списке).

**Примечание:** Данные предназначены только для информационных целей. Не используйте для обхода ограничений. Всегда проверяйте актуальность на официальных источниках.
